# Student Progress Tracker (Mini Project) - Version 1

A Python and SQLite-based mini project designed to track student study progress, manage student records, monitor daily learning activities, and analyze study performance through database-driven operations.

## Features

- Add Student
- Add Daily Progress
- Auto Date Generation
- View Student Progress
- View Students
- Calculate Total Study Hours
- Calculate Total DSA Problems Solved

## Technologies Used

- Python 3
- SQLite3
- Git
- GitHub
- Visual Studio Code

## Database Design

### Student Table
- Student ID
- Student Name

### Progress Table
- Progress ID
- Student ID
- Date
- Study Hours
- DSA Problems Solved
